My main web page looks seems like a poster -- a beating heart as the entrance for the next web page, and using some attribute tags and circuits to illusacte the characteristic of programmer :).

The image gallery pages on my site will be based on this design which I want to be the following...

According to four principles, proximity, alignment, repetition, contrast. I made the following table. In terms of Proximity, related items should be placed near each other and look organized. I separated the yellow images and red images which represents normal member and VIP respectively. Besides, the whitespace helps separate different subjects. As can be seen from the table, the clear lines separates different models on the page. Text and images line up with other text and images. How about repetition? I use the same font, same rule for the entire page and use a consistent colour for the same model. All these designs create a unify presentation. Finally is the contrast. I use really different colour standout from the background. And different typefaces, font size to create contrast.
